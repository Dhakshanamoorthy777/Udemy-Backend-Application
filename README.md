# 🎓 Udemy Clone Backend Application

A RESTful backend application inspired by the Udemy online learning platform, developed using **Java**, **Spring Boot**, **Spring Data JPA**, **Hibernate**, and **MySQL**.

## 🚀 Features

- User Registration & Login
- Course Management (CRUD)
- Section & Lecture Management
- Course Enrollment
- Course Reviews
- Order & Order Item Management
- Course Search
- Course Pagination
- DTO-Based Request & Response
- Global Exception Handling
- Input Validation

## 🛠️ Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- ModelMapper
- Lombok
- REST API
- Postman
